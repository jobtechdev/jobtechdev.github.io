Hack for Sweden

The Hack summary:

Hack for Sweden kicked off in a racing start with API, AI and QnA sessions. Then the hackers started to code!

For us the best part was seeing that our datasets were in the top 3 most popular APIs to use! Seven different teams used our APIs to build smart solutions for the good of the Swedish labour market.

We are happy to present the seven teams solutions, and here they are!

Team-The breakfast club

We present a service that through user input suggests related occupations to their skills and suggesting meet-ups events related to their interests. This is done through combining Arbetsförmedlingen APIs (ontology and Job Forecast) and an open source API from Meetup.com

Datasets: Ontology API (Arbetsförmedlingen) + Job Forecast (Arbetsförmedlingen).

Breakfast Club

Team-Hack From Sweden

A developed “Smart Cash Index” (SCI) for plotting housing cost against remote jobs. The SCI will also show quality of life in proximity to parks, hospitals, schools etc. This is done to inform users of good places to live while working remotely in Sweden.

Datasets: Platsbanken API (Arbetsförmedlingen) + Living costs (SCB)

Hack from Sweden

Team-Ready Work

Using Microsoft Azure's translation API to create a more reliable and stable translation of Arbetsförmedlingen's data compared to what is presently done, opening it up to be more conveniently used regardless of the user's present language proficiency.

Instead of simply listing the region with the Most job listings for any given skills, the goal is to provide a ranking of where the user's unique skill set is the most needed based on amount of ads compared to population size. This ranking can be further developed to include the user's own preferences as a filtering method.

Ready Work then presents the user with relevant information on the region as well as a listing of job openings matching their personal set of skills.

By also providing a translation service for the job application process itself, Ready Work can help connect the user with employers on a more even playing field.

Datasets: Platsbanken API (Arbetsförmedlingen) + Translator API (Microsoft)

Ready work

Team-Map Your Career

This is a tool that can be used in connection with an on-person visit to employment office or remotely. The newcomer to Sweden can enter some of his skills and relevant professions are displayed in a bubble chart. Most relevant jobs are shown in the center and highly related ones around. Hovering over the professions displays a description and a trends forecast and allows to click on those that you are interested in. At the last stage, the officer can gather some basic background information and add personal comments. Finally, a summary report is provided in a printer-friendly format.

Map your career facilitates the discussion on career options providing dynamic content in an easy to understand way.

Datasets: Historical job postings, professions, Ontology and Forecasts (Arbetsförmedlingen)

Team-Match me if you can

When selecting the topics of interest our software is initiating a lookup in the graph database Neo4J that was initially fed with all information about the available APIs. The result of this lookup will be a sorted list of APIs that are related to each other and contain data about the respective topics. These APIs can simply be combined for building the software of your dreams.

Datasets: All datasets?

Team-Match Yourself

By submitting the questions at hand you get presented with the top jobs that match your personality, it is that easy! The service let's you walk through a proven scientific method for understanding the personal traits by asking 20 question and by combining Arbetsförmedlingen APIs (ontology and real-time job api) we collect information about matching words used to describe certain job categories to get a personality score of a job to be match with. This is futher analysed to get the top matching jobs for the person at hand. Personality to job.

Datasets: Ontology API + Platsbanken API (Arbetsförmedlingen)

Match yourself

Team-LocationRank

Entering the job market in Sweden might be sometimes challenging, whether you're a job seeker or an entrepreneur. Some areas might suit your better than others based on a set of factors, such as recruitment market size or relative safety of a certain area.

Here comes the problem of finding the best geographical area for you to operate, both as an employee or employer.

We believe there is potential for facilitating this kind of decisions, which our app can help you with.

Leaveraging an Arbetsförmedlings API this app produces a comparison of two geographical areas based on certain competencies. This will help both employees and employers to find the best geographical area for their business.

Datasets: Ontology API (Arbetsförmedlingen)

Jobtech Forum

Välkommen till ett nytt Jobtech Forum den 13 oktober 2017!

Arbetsförmedlingens avdelning Digitala tjänster är glada att få bjuda in dig till höstens Jobtech Forum!

I våras genomförde vi ett välbesökt första Jobtech Forum. Nu vill vi fortsätta dialogen om hur vi gemensamt kan skapa förutsättningar för digitala samarbeten inom området jobbmatchning. Vår vision är ett ekosystem där olika aktörer delar en gemensam infrastruktur som gör det enklare för den som söker arbete eller vill rekrytera. Genom att samarbeta kan vi ta tillvara innovationskraften och bidra till en effektivare matchning på arbetsmarknaden!

Under dagen kommer vi att presentera flera nyheter bland annat en första version av Job store, Jobskills (kartläggning av kompetenser för målgruppen nyanlända) samt öppen ontologi (realtidsuppdaterat värdeföråd av kompetenser). Kan man räkna fram vilket företag som kommer att rekrytera? I samarbete mer RISE har vi tagit fram en första version av tjänsten som kombinerar historiska platsannonser med ekonomisk data.

Detta och mycket annat vill vi diskutera mer dig!


Lokal: Epicenter, Mäster Samuelsgatan 36 i Stockholm

Preliminärt program:

08.30 Registrering och kaffe

09.00 -11.00 Presentation av vision, vad som hänt sen sist och hur vi går vidare. Erik Sandström, chef för Matcha direkt, Arbetsförmedlingen

11.00-1200 Keynote speaker, Pôle Emploi, den franska statliga arbetsförmedlingen berättar om Emploi Store och deras samarbete med privata aktörer för att effektivisera matchningen på arbetsmarknaden.

12.00- 14.00 Lunch, samtidigt kan du ta del av utställningar om pågående utvecklingsinitiativ och delta i open space-diskussioner

Arbetsförmedlingen visar nya och kommande tjänster. Här kan du också diskutera affärsmodeller och samarbeten i mindre grupp. I år har vi har flera av våra skarpa utvecklare på plats för att diskutera API:er, öppna data och gemensam utveckling baserat på öppen källkod.

Dagen vänder sig till personer i ledande befattningar inom företaget och personer som arbetar med teknikutveckling. Antalet platser är begränsat.
